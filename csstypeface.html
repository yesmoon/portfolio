CSS Typeface

Experimental Typeface Development

How do you create a typeface that can only live on the web? To build the letterforms, I used the viewport as an empty page in which I would draw the letterforms in opaque black shapes that would render the letterforms. With certain letterforms, white blocks were used to hide parts the CSS shapes, creating the letterform through a reductive process.

After all the letterforms were constructed, I turned all the CSS shapes into transparent color blocks that would reveal the construction of the letterforms and its process.
